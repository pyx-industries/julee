Sphinx HCD
==========

.. define-accelerator:: sphinx-hcd
   :status: active

   Human-Centered Design documentation extension for Sphinx. Provides
   directives for defining personas, journeys, epics, stories, applications,
   and integrations with automatic cross-referencing and validation.

   Located at ``src/julee/docs/sphinx_hcd/``.

   **Capabilities:**

   - Define document-first entities (personas, journeys, epics, stories)
   - Parse code-first entities (apps from YAML, stories from Gherkin)
   - Generate index pages and relationship diagrams
   - Validate documentation coverage at build time
   - RST repository backend for lossless round-trip editing
