Solutions
=========

A **Julee Solution** is a software system built on the :py:mod:`Julee framework <julee>`.
See :py:mod:`julee.core.entities.solution` for core concepts.

Key entities:

- :py:mod:`~julee.core.entities.solution` - Solution structure and organisation
- :py:mod:`~julee.core.entities.pipeline` - Pipeline concepts (use cases + Temporal)
- :py:mod:`~julee.core.entities.application` - Application types and patterns
- :py:mod:`~julee.hcd.entities.accelerator` - Accelerators (business capabilities)
